# Proyecto Final Full Stack

## Descripción
Aplicación web desarrollada como proyecto final del grado de Desarrollo de Aplicaciones Web.
Permite a los usuarios registrarse , iniciar sesión y gestionar su perfil, mientras que el administrador puede realizar operaciones CRUD sobre la base de datos.
El proyecto integra frontend y backend, conectando la aplicación con una base de datos MySQL mediante PHP.

## Tecnologías utilizadas
  - HTML5
  - CSS3
  - JavaScript
  - PHP
  - MySQL
    
## Funcionalidades
  -Registro de usuarios
  -Inicio de sesión
  -Conexión a base de datos
  -Operaciones CRUD (crear, leer, actualizar y eliminar datos)
  -Panel de administración
